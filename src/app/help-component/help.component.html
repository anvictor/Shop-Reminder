The most convenient way to enter or sort a To-Do list or purchases.

  1. Paste and copy of text, or write or dictate the list of items, goods or tasks you want to do.
    Separate it with commas (,). In the numbers instead commas use only decimal dots.
  2. A list of cases (or purchases) will be created at the bottom.
    All elements will look like colored stripes.
    Cross out the Done items by pressing the red cross (X).
    The second click on the cross (X) will make the element not crossed out again.
  3. You can sort elements of similar classes by 8 color groups.
    By clicking the 8 color striker below, assign same colors to similar items.
    You can reassign again any order class to crossed out element.
  4. The first element will be used as the title of the list.
    EXAMPLE:
    "buy milk, plant a tree, build a house"
    The element before the first comma "buy milk," will be the Name of this list.
    At the same time it will be the first list element.
    If your device has a speech recognition feature, say the word "comma" after each phrase.
    Use numbers with a decimal point, not a comma.
  5. You can change the Name of the list manually.
    Write at the beginning of the text the following:
    Name: Your NewName
    put comma and continue write the list.
  6. Clicking on the (Old) button will open the list of old lists.
    To delete an old list, press (X).
    To use or edit the old list, click (ok).
    If you want to make a copy of the old list, change the "Name: _name," to another.
    Both lists will remain, with the old and the new name.

Найзручніше введення і сортування списку справ або покупок.

  1. Вставте копію тексту, напишіть або Надиктуйте предмети, товари, або справи, які ви хочете виконати.
    Розділіть їх комами (,). В числах використовуйте не коми, а точки.
  2. Якщо ваш пристрій має функцію розпізнавання мовлення, вимовляйте слово "кома", після кожної фрази.
    Іноді функція роспізнає тільки "запятая".
    Це залежить від вашого налаштування. Використовуйте замість чисел з комою, числа з крапкою.
  3. Внизу сформувався список справ (або покупок).
    Всі елементи матимуть вид кольорових планок.
    Закреслюйте виконані пункти, натискаючи червоний хрестик (Х).
    Друге натискання на хрестик (Х) зробить елемент знову не закресленим.
  4. Можна елементи схожих класів впорядкувати за 8 колірними групами.
    Натискаючи кнопки з номерами кольорів внизу, привласніть подібним елементам однакові кольори-класи.
    Закресленому елементу можна знову присвоїти будь-який клас сортування натиснувши кольорову кнопку.
  5. Перший елемент буде використано в якості заголовка списку.
    НАПРИКЛАД у тексті:
    «Вітання, Слава Україні!, Героям Слава!»,
    Елемент, за яким стоїть перша кома «Вітання,» буде назвою цього списку і одночасно першим елементом.
  6. Ви можете змінити назву списку вручну. Для цього на початку тексту напишіть таке:
    Name: ВашаНоваНазва, поставте кому і далі продовжуйте список, як вам буде зручно.
  7. Натискаючи на кнопку (Old), ви відкриєте список старих списків.
    Для видалення старого списку натисніть (Х).
    Для використання, або редагування старого списку натисніть (ok).
    Якщо потрібно зробити копію старого списку змініть йому назву Name: на іншу.
    Збережуться обидва списки, із старим та з новим ім'ям.

Наиболее быстрый ввод и удобное редактирование списка дел или покупок

  1. Вставьте копию текста, напишите или надиктуйте предметы, товары или задачи, которые вы хотите исполнить.
    Разделите их запятыми (,). В числах используйте десятичную точку вместо запятой.
  2. Если устройство имеет функцию распознавания речи, произносите слово "запятая", после каждой фразы.
    Иногда функция распознает только "кома".
    Это зависит от вашей настройки.
  3. Внизу сформировался список дел (или покупок).
    Все элементы будут иметь вид цветных планок.
    Зачеркните выполнены пункты, нажимая красный крестик (Х).
    Второе нажатие на крестик (Х) сделает элемент опять не зачеркнутым.
  4. Можно элементы подобных классов отсортировать по 8 цветовыми группами.
    Нажимая кнопки с номерами цветов внизу, присвойте подобным элементам одинаковые цвета-классы.
    Меняя у зачеркнутого элемента класс цветовой сортировки, вы делаете его не-зачеркнутым.
  5. Первый элемент будет использован в качестве заголовка списка.
    НАПРИМЕР в тексте:
    «Я сознаю, что аннексия Крыма и захват Донбасса, это беззаконие»,
    Элемент за которым стоит первая запятая «Я сознаю,» будет названием этого списка и одновременно первым элементом.
  6. Вы можете изменить название списка вручную. Для этого в начале текста допишите следующее:
    Name: ВашеНовоеНазвание
    поставьте запятую и дальше остальной список.
  7. Нажимая на кнопку (Old), вы откроете список старых списков.
    Для удаления старого списка нажмите (Х).
    Для использования, или изменения старого списка нажмите (ok).
    Если нужно сделать копию старого списка измените его название Name: на другое.
    Сохранятся оба списка, со старым и с новым именем.
